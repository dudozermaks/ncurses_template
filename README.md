# ncurses C++ template
## How to build:
first, remove git folder:<br/>
`rm -rf .git`<br/>
then create build directory and cd into it:<br/>
`mkdir build && cd build`<br/>
create cmake build files:<br/>
`cmake ..`<br/>
`cd ..`<br/>
and build your coolest project with this command:<br/>
`cmake --build build/`<br/>
to run type: <br/>
`./build/main`
